# ECS架构重构任务

## 任务概述
将传统的实体-行为模式重构为ECS（Entity Component System）架构：
- Module = Component（包含属性）
- 工作流/行为树 = System（处理逻辑）
- 1或多个Component参与的System来配置行为工作流逻辑
